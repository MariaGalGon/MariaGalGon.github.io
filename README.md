## Información básica

A continuación se ven los tipos de letra que se pueden usar en GitHub Pages.

```markdown

- **Negrita**
- Cursiva
- `Código`
- [Enlace]

```

Para realizar una cita:
> Se escribe así
 
Para poner tareas se puede hacer así:
- [ ] Tarea incompleta
- [x] Tarea completada
